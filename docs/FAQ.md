# FAQ

## Is this a product repo?
It is a public research repo with a bounded smell result. The result is real on the declared benchmark surface, but the repo does not make a product claim.

## What does "surrogate receptor-response panel" mean here?
The smell surface in this repo is a fixed synthetic panel used to test a bounded encoding question. It is not a claim about the full empirical receptor panel.

## Why is the repo public if the scope is narrow?
Because the result is reproducible and honest on that narrow surface. Public audit is useful even when the scope stays bounded.

## Does this repo claim broad smell understanding?
No. The repo claims only what is measured in the committed proof artifact.

## Why does the repo mention the certified subset?
To stay honest about scope. This repo does not claim that the current smell result is admitted to that broader set.
